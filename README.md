# Simple Scaffolder tool

I did a project before using C++, now I did it again using python, and it was really straightfoward.

## Usage

```bash
python3 scaffold.py {project_name}
```

## Arguments

* -t TEMPLATE : Specifies the template to use from the templates folder. Default = "Basic"

```bash
python3 scaffold.py {project_name} -t {template_name}
```
* --templates : Shows a list of all the templates available:

```bash
python3 scaffold.py --templates
```
This will display:
```bash
Available templates:
- Basic 
```

## Notes

Doing this file management using python was trivial compared to doing it in c++
